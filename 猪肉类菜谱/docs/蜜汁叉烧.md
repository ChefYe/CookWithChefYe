# 干蒸排骨（烤箱版，腌制10分钟以上，一小时为佳）

![蜜汁叉烧](/images/蜜汁叉烧.jpg)

## 步骤

叉烧配方（家庭版，腌制12小时以上）
1️⃣腌制：
五花肉1000g 糖83g 老抽20g 叉烧酱60g 
诸侯酱10g海鲜酱10g 红腐乳30g 蚝油10g 
玫瑰露酒30g 玉米淀粉15g 油15g 
加适量红葱头 大蒜 姜 抓拌2分钟

2️⃣蜜汁：
麦芽糖100g+水20g+红醋10g

3️⃣烤制：
200度（392华氏度）
烤15分钟正面 15分钟反面
烤前 和 中途刷蜜汁  

4️⃣叉烧淋汁：腌肉剩下的料汁，
再加糖10g 油15g 老抽10g 水160g，
小火收浓再过滤，酱汁就做好啦
